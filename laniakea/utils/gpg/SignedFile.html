    Class SignedFile

    Handle files signed with PGP

    The following attributes are available: contents - string with the content (after removing PGP armor) valid - Boolean indicating a valid signature was found fingerprint - fingerprint of the key used for signing primary_fingerprint - fingerprint of the primary key associated to the key used for signing
